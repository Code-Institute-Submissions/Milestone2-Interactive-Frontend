How to play this game

  • This game is intended to train your memory skills. It is being played by two players.

  • Please register two players via the button "Register players".

  • Select then the size of playfield and hit the start button. By default, it is set to 8 cards. You might also want to try fieldsizes of 16 or 36 cards.

  • Each player tries to find the maximum amount of pairs. To do so, each player clicks two cards per round. If you succeeded and have found a match, the same player can continue and try next. If no pair was found, the other player is next. The players who has found the most pairs, wins the game.

  • To start over a game, stop the current game and select playfield size.

    Have fun and a good memory! :)
